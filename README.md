# BlackJackJS

A black Jack Game made with vanilla JS, Webpack and SASS.

## Installation

First install all the node modules. 

```bash
npm i
```

Once done, run the Webpack Server. 

```bash
npm run start:dev
```

Go to your favorite browser @ http://localhost:8080/ and give it a shot!
